# Smart Lecture AI

تحويل محاضراتك الصوتية إلى ملخصات، أسئلة، وخرائط ذهنية ذكية.

## كيفية النشر على Streamlit Cloud

1. تأكد من تثبيت كل المكتبات:
   ```
   pip install -r requirements.txt
   ```
2. ضع المفاتيح السرية في `.streamlit/secrets.toml`
3. ادفع المشروع إلى GitHub
4. افتح https://share.streamlit.io/ واختر Repository للـ Deploy
5. اضغط Deploy وسيكون التطبيق جاهزًا

## مميزات التطبيق

- تسجيل مستخدمين (Auth)
- رفع المحاضرات وحفظها
- قيود الاستخدام (Usage Limits)
- الدفع عبر Stripe (Student / Pro)
- Landing Page جذابة
